JavaScript

Programming Language introduced to make the WebPages alive.

Scripts are made to give actions to various elements in the WebPage. They are executed as Plain text as they don't need any compiler like C++ or Java.

It is Very Different from Java as the language is based on the specifications of ECMAScript.

Javascript can do like

  • React to user actions, run on mouse clicks, pointer movements, key presses.
  • Get and set cookies, ask questions to the visitor, show messages.
  • can remember the data as in a local storage.
  • Can also Interact in the network by sending requests over the network by using server like Node.Js .
